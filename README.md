# Getting Started

## Linux Systems

```BASH
git clone https://github.com/EmbeddedSystems-UniversitySiegen/eclab-task3.git
cd eclab-task3
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Refer to the task documentation for the details. 

## Tests

Run tests using pytest.

``` BASH
pytest
```
